# Song File Format
Synthax `time_before_trigger, key_to_be_triggered`
```
time,key
```
Example (21 Keys)
```
500,n
500,c
500,a
500,m
1000,b
```

Example (36 Keys)
```
500,n
500,c
500,a
500,m
500,SHIFT+b
500,CTRL+m
500,SHIFT+b
500,CTRL+m
```
